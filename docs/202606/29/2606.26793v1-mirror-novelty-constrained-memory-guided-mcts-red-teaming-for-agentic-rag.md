## 摘要
多模态智能体检索增强生成（RAG）系统将攻击面从提示注入扩展到包括文本投毒、图像注入、直接查询攻击以及协调器级工具操纵。现有红队方法通常只针对特定攻击面，且常重复使用已知攻击模板；在文本投毒基准上我们测得73–84%的精确重复率。我们提出MIRROR，一个统一的跨攻击面框架，其在显式新颖性约束下，以检索到的上下文为条件生成候选，并执行记忆引导的蒙特卡洛树搜索。确定性新颖门通过归一化比较拒绝任何与检索集匹配的候选，使检索能为搜索先验提供信息而不会引起提示复制。在多模态智能体RAG目标的四个攻击面上，MIRROR在图像投毒上达到76%的攻击成功率（基线为52%），在协调器攻击上以一半的查询成本达到97%的攻击成功率，并具有最低的跨攻击面方差（变异系数0.47）。相比之下，专用基线方法跨攻击面性能崩溃：后缀优化在文本投毒上达到79%攻击成功率，但在直接查询上仅为1%。我们发布ART-SafeBench，包含41,815条包内记录和运行时适配器，在四个攻击面上总计得到41,991+条记录。

## Abstract
Multimodal agentic retrieval-augmented generation (RAG) systems expand the attack surface beyond prompt injection to include text poisoning, image injection, direct-query attacks, and orchestrator-level tool manipulation. Existing red-teaming approaches are typically surface-specific and often recycle known attack templates; on text-poisoning benchmarks we measure 73-84% exact duplication. We present MIRROR, a unified cross-surface framework that performs memory-guided Monte Carlo tree search while conditioning candidate generation on retrieved context under an explicit novelty constraint. A deterministic Novelty Gate rejects any candidate matching the retrieval set under normalized comparison, allowing retrieval to inform search priors without enabling prompt copying. Across four attack surfaces on a multimodal agentic RAG target, MIRROR attains 76% ASR on image poisoning compared with 52% for baselines, 97% ASR on orchestrator attacks at half the query cost, and the lowest cross-surface variance (coefficient of variation 0.47). In contrast, specialized baselines collapse across surfaces: suffix optimization reaches 79% ASR on text poisoning but 1% on direct queries. We release ART-SafeBench with 41,815 in-package records and runtime adapters yielding 41,991+ total records across four surfaces.

- **关键技术细节**：
  - **双阶段架构**：包含一个由成功攻击痕迹组成的**情景记忆库**（Prior Network）和一个新颖性约束的规划器。记忆库通过`k‑NN`检索给出操作先验和每个实例的拒绝集合。

## 六、主要结论与发现

- **跨表面一致性**：MIRROR是唯一能在全部四个攻击面上端到端运行的方法，跨表面ASR变异系数最低（0.47），而专用方法如GCG在B1达79%，B3仅1%，表现出“表面崩塌”。
- **显著性能与效率**：
  - 图像投毒(B2)：76% ASR vs. 基线52%（OV）和32%（LSB）。
  - 协调器攻击(B4)：97% ASR，且查询效率是Toolflip的两倍（Q/Success 1.00 vs 2.08）。
  - 直接查询(B3)达31%，在所有评估方法中最高。
- **新颖性的决定性作用**：种子池精炼方法（PAIR, TAP, PS）B1的ASR高达58‑77%，但其中73‑84%为基准重复，真实新颖攻击成功率仅6‑9%。MIRROR通过新颖门实现0%基准精确重复，Novel‑ASR等于原始ASR（47%）。
- **对补丁策略的鲁棒性**：在已知攻击被防御的压力测试下，MIRROR保持低内部重复，而其他精炼方法发生严重自我重复（93‑97% SelfDup），表明新颖性约束对持续漏洞发现有重要价值。
- **基准贡献**：发布的ART‑SafeBench提供了大规模、多表面、可扩展的评估标准，支持后续研究。

## 七、优点

- **方法论创新**：将检索记忆、MCTS规划与硬性新颖性约束深度融合，形成闭环，有效防止“检索即复制”的失败模式，并能跨表面迁移。
- **双重验证机制**：强制最终目标回放，将探索性仿真与确认性执行分离，降低了仿真‑真实差距带来的成功率虚高，使评估更可信。
- **诊断性指标全面**：引入DupBench、Novel‑ASR、SelfDup等指标，揭示基线方法的虚假高分，引导社区关注真正的攻击发现而非模板重放。
- **基准完整性**：ART‑SafeBench覆盖四个典型攻击面，包含明确的版本管理、许可分割（Core vs Extended）和领域偏移设置，提高了可复现性与实用性。
- **压力测试视角**：通过已知攻击池补丁实验，模拟了攻击‑防御动态，展示了方法在迭代对抗环境中的潜力。

## 八、不足与局限

- **新颖性定义的局限性**：新颖门仅基于精确文本归一化匹配，无法检测语义等价、模糊同义或风格变体攻击，新颖性证书的范围较窄，可能存在语义过准绕过的问题。
- **部分表面ASR非最优**：在文本投毒(B1)上，专用后缀优化方法GCG取得79%的更高ASR（尽管基于重放且无新颖保证），MIRROR的47%并非最高绝对成功率。论文将自身定位为跨表面规划器而非单表面强者，但在特定场景下竞争力有限。
- **领域偏移敏感**：CYBER RAG案例显示，在严格结构化输出和领域偏移下，基线反而胜出，MIRROR的检索先验可能受语料‑受害者对齐度的影响较大。
- **图像表面评估不足**：B2的新颖性指标因缺乏图像相似度度量而缺失，仅依赖文本度量，对图像攻击的新颖性评估不完整。
- **资源依赖未量化**：实验依托商用LLM API，成本、延迟未报告，缺少对本地硬件或开源模型适用性的讨论，实际部署可行性存疑。
- **消融研究有限**：未单独分析先验网络、新颖门或重放验证等各个模块的贡献，仅作为一个整体系统呈现。
- **攻击面定义的静态性**：四个攻击面是预定义的，未来可能出现新的工具链或模态攻击，框架的可扩展性未讨论。
